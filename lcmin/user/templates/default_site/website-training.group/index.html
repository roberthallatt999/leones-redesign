Common Terminology

BACKEND/CONTROL PANEL
The web-based interface where the site’s content is managed.

FRONTEND
The public interface that is visible when someone visits the website.

CHANNEL
The database container that houses a certain type of content.

ENTRY
Any type of content that is editable in the backend.

CREATE
Create a new entry.

EDIT
Allows you to change an existing entry.
